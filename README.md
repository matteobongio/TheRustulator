# The Rustlulator

A rust port of [Themulator](https://github.com/Bgabri/Themulator)


```
Usage: thrl <COMMAND>

Commands:
  login     Set Login information
  download  Download Test Cases
  help      Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version

```
